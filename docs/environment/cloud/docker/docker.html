Docker

简介

Docker是一个开源的容器引擎，在17.03版本后主要分为CE(社区版)和EE(企业版)两种版本。它的官方网址是https://www.docker.com/。

架构视图

安装使用

这里只描述在linux centos上安装使用。

安装

Docker的安装较简单可以通过以下命令完成：

yum install -y yum-utils device-mapper-persistent-data lvm2
yum-config-manager --add-repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo
#可以通过以下命令行选择需要安装的版本
yum list docker-ce --showduplicates | sort -r
#需要将其中的VERSION_STRING替换为你需要安装的版本
yum install docker-ce-<VERSION_STRING> docker-ce-cli-<VERSION_STRING> containerd.io
常用命令
#启动
systemctl start docker
#镜像查看
docker images
#镜像下载
docker pull ***
#镜像删除
docker rmi ***
#查看正在运行的容器
docker ps
#查看所有容器
docker ps -a
#容器启动
docker start ***

容器启动后，通常会进行定制化改造，一般可以通过以下命令进行操作

docker exec -it *** /bin/bash
#修改完相关若重启(restart)失败，可以查看相关日志
docker logs ***
#若启动失败，可以在以下路径搜索相要修改的文件
/var/lib/docker/overlay2
